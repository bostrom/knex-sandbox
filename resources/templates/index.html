Bookends

A Node.js module for easily hydrating complex trees of SQL data.

Bookends is a small library that sits on top of the Bookshelf.js ORM that makes querying for a tree of SQL data straightforward. A simple string indicates which columns, relations and aggregations you want to hydrate from the database. 

bookends.hydrate(Author, '[first_name,last_name,books=[title]]').then(function(result) {
  console.log(result.records[0].first_name);     // Louisa May
  console.log(result.records[0].last_name);      // Alcott
  console.log(result.records[0].books[0].title); // Little Women
});
